Introduction:


This Program was written in 2001, the year I went to Java One and accepted the Dutch JUG challenge to create a program in 5 days that calculates all possible solutions to the Yasumi puzzle.
The ones who tried this beform me know that writing this program could be faster than actually executing it.
Due to the numerous possibilities of placing the building blocks, a brute force program would take longer than one hour to come to a result.
After some thinking and programming I came with this program and during the load, you will get an introduction on how this program is operating.
The time to load all the possible solutions that this program is capable of finding will be under 2 Minutes where the elapsed time back then was around 15 minutes.
